# Contributing to Hanif AI Architecture

We're glad you're interested in contributing! This project aims to build a more ethical and conscious AI architecture.

## How Can I Contribute?

### ⚖️ Adding Ethical Principles
Our "Artificial Conscience" layer is only as good as its training data. You can contribute by adding universal ethical principles or case precedents to `src/ac_layer/ethics_data.json`.

### 🧠 Enhancing the Mind Logic
If you have ideas for better weighing functions or decision-making algorithms, please open a Pull Request affecting `src/am_layer/orchestrator.py`.

### 🧪 Improving Benchmarks
Help us test the system by adding new moral conflict scenarios in `tests/test_moral_conflicts.py`.

## Pull Request Process

1. Fork the repository.
2. Create your feature branch (`git checkout -b feature/AmazingFeature`).
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`).
4. Push to the branch (`git push origin feature/AmazingFeature`).
5. Open a Pull Request.

## Code of Conduct
Please be respectful and ensure that your contributions align with the project's goal of preserving human dignity and truth.
